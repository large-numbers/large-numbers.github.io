About this site

Welcome to Adesdae's Large Number Site, a site where I attempt to define the biggest numbers ever. Please note that this is not the first site on the internet to define large numbers. The earliest I have seen was Robert Munafo's large number webpage, dating back as far as 1996. Another example was Jonathan Bowers' site, created in 2002. Munafo was primarily interested in explaining the certain notations generating large numbers, while Bowers was primarily interested in making his own system to push the theoretical limit of numbers created by humanity. My site is the first to merge these two topics into one site.

Site layout

The site is comprised of five sections, each section discussing a certain topic. Currently there is only one section.

Section 1 will explain how ordinals and the fast-growing hierarchy work, as well as certain functions to generate ordinals.

Section 2 will explain how some large number notations of others work, as well as some of my extensions to those systems.

Section 3 will introduce some of my own work, including my array function known as Adesdae's Array Notation (AAN), and some numbers defined with it.

Section 4 will compare my array notation to the fast-growing hierarchy, as well as some other people's work.

Section 5 will contain a complete list of numbers, some I coined, some I did not.
