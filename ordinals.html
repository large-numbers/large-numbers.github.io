Introduction to ordinals

First of all, what is an ordinal? An ordinal is a type of infinity. So how can there be more than one of them? They're from set theory where infinity + 1 does not equal infinity, but 1 + infinity does. Why can't it be more helpful? Because.

An ordinal is defined as the set of all smaller ordinals. The smallest possible ordinal, 0, is defined to be the empty set ({}), since no ordinal is smaller than 0. We then define the ordinal 1 = {0} = {{}}. Next, we can define 2 = {0,1} = {{},{{}}}. We can continue with 3 = {0,1,2} = {{},{{}},{{},{{}}}}, 4 = {0,1,2,3} = {{},{{}},{{},{{}}},{{},{{}},{{},{{}}}}}, etc. but the ordinal system continues with the first transfinite ordinal ω. ω is the first ordinal above all finite ordinals. It is also equal to {0,1,2,3,...} = {{},{{}},{{},{{}}},{{},{{}},{{},{{}}}},...}. We can then define ω+1 as the set containing all ordinals before ω and then ω. We can also define ω+2 as the set containing all ordinals before ω+1 and then ω+1. Let's define S(α) as the ordinal "above" α. In other words, it is the set containing all the members of α and then α. This can be more formally written as α∪{α}.

We can continue by defining ω2 as the supremum of ω+1, ω+2, etc. ω3 as the supremum of ω2+1, ω2+2, etc. We can also define ω2 as the supremum of ω2, ω3, etc. ω3 as the supremum of ω22, ω23, etc. We can also define ωω as the supremum of ω2, ω3, etc. ωωω as the supremum of ωω2, ωω3, etc. Eventually we reach ωωω..., known as ε0.
